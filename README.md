# SRE Automation Toolkit 🛠️

Welcome to my SRE and DevOps automation repository. This project serves as a centralized hub for scripts, tools, and configurations I use to automate repetitive server administration tasks, standardize infrastructure, and eliminate operational toil.

## 🎯 Objectives
* **Automate Toil:** Scripts to handle routine OS configurations and fix common drifts (e.g., strict NTP/Timezone alignment across Linux servers).
* **Enhance Observability:** Ensure servers are properly configured to play nicely with monitoring stacks like Zabbix, Prometheus, and Grafana.
* **Standardization:** Maintain a consistent and reliable state across all servers, from home lab environments to production.

## 🚀 Contents
* `/os-config` - Base operating system configuration scripts (e.g., `fix-timezone.sh`).


## ⚙️ Usage
Each script contains its own comments and execution requirements. Always review the code before running it in a production environment.
